# PROJECT TITLE :
 SMART-BUS-ATTENDANCE-SYSTEM-
# PROJECT OVERVIEW :
Smart Bus Attendance System is an application developed to automate the recording of bus arrival (in-time) and departure (out-time). In many places, these details are entered manually, which can lead to errors and consume time.
This system allows bus timing information to be recorded digitally through a mobile application. The data is stored in a database and can be viewed by the administrator whenever required. The system helps improve accuracy, reduce paperwork, and make bus monitoring easier.
# PROBLEM STATEMENT :
Manual recording of bus in-time and out-time is time-consuming and prone to errors. This project aims to automate bus attendance and timing records, improving accuracy, efficiency, and data management.

# PROJECT OBJECTIVES 
  → To automate the recording of bus in-time and out-time.
  → To reduce manual data entry and human errors.
  → To store bus timing records securely in a database.
  → To provide easy access to attendance and timing information.
  → To improve the efficiency of bus monitoring and management.
